# Sitcom Humor Lab

## Goal

We try to use different toolkit to analyze sitoms by combining:
- **audience ratings**
- **transcripts and subtitles**
- **text analysis**
Our initial case study is the populer sitcom *The Big Bang Theory* (TBBT).
